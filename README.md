# Philo Vault

Obsidian 个人哲学知识库，与姊妹库 `valu_air`（投研）同范式：**PARA 工作层（散文）+ `wiki/` 知识本体层（编译产物）**，由 auto-wiki v2 引擎驱动。

- 两大方向：**理论哲学**（metaphysics 形而上学 / epistemology 认识论）+ **实践哲学**（ethics 伦理学），理论是实践的奠基层（跨域签名边 `grounds`）。
- 编译单向：`Inbox(散文思考/讨论) → /auto-wiki ingest → wiki(结构化本体)`。
- 每域一个 `data.db`（双时态 SQLite）；6 档时间模型；退役不删除——**个人立场（T1 facts）的切换轨迹是本库最有价值的时间序列**。

入口：`00-Dashboard/Dashboard.md`（仪表盘）· `wiki/_index.md`(域注册表) · 各域 `_ontology.md`(本体契约，metaphysics 为母本)。

Agent 指令见 `CLAUDE.md`。常用命令：`/auto-wiki recall|ingest|query|lint`、`/digest-inbox`、「执行今日例行任务」。
