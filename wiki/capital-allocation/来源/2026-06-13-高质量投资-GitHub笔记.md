# 来源摘要：《高质量投资》GitHub 最佳笔记

> 二手读书笔记（非原书全文），全文留存于 `Inbox/Quality Investing（高质量投资）最佳笔记.md`。

## 覆盖内容
- **Building Blocks**：资本配置（growth capex / R&D+A&P / 并购 / 分红回购 / 营运资本）、
  [[资本回报]]（资产周转×利润率×现金转化）、[[增长来源]]（份额/地域/价格组合/周期/结构）、
  好管理层（纪律管家、独立长期坚韧、低调、用人、坦诚）、行业结构（迷你垄断/局部垄断/寡头/进入壁垒与理性）、
  客户利益（无形/保证/便利）。
- **Patterns（[[竞争优势模式]]）**：[[经常性收入]]、友好中间人、收费公路（金标准/魔法成分）、
  低价加成、[[定价权]]、[[品牌力]]、创新主导、前向整合、份额获取者、全球能力、企业文化、复制成本。
- **Pitfalls（[[质量投资陷阱]]）**：周期性、技术颠覆、依赖性（政府/集中度/新进入者）、客户偏好迁移。
- **Implementation**：对抗短期主义；买入错误（自上而下侵入/下周一乐观/过度自信/杠杆）；
  持有错误（温水煮蛙/论点漂移/会计红旗/禀赋效应）；质量优先、估值其次。
