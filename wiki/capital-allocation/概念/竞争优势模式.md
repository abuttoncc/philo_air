# 竞争优势模式

《高质量投资》归纳的十二种让公司达成质量状态的模式（Patterns）。
没有严格规则，模式是组合拳：

| 模式 | 一句话 |
|---|---|
| [[经常性收入]] | 存量客户持续买服务/耗材，现金前置 |
| 友好中间人 | 牙医/安装工/顾问替你说话，专业培训即锁定 |
| 收费公路 | 小而不可绕过：金标准（评级/检测）、魔法成分（酶/香精/工业气体） |
| 低价加成 | 低价+特色（IKEA/Costco/Ryanair），千刀万剐式累计成本优势 |
| [[定价权]] | 真实定价权多是有条件的；按价值定价（孟山都与农户分成） |
| [[品牌力]] | 传承不可复制；信任与一致性；规模反击 |
| 创新主导 | 高毛利→更大研发/广告投入→良性循环；增量创新优于豪赌 |
| 前向整合 | 强品牌自控渠道，攻守兼备；特许经营费=品牌经济力 |
| 份额获取者 | 与宏观脱钩的增长 |
| 全球能力 | 本土霸主若海外有更强同行则不安全 |
| 企业文化 | 与战略匹配的文化自我延续；家族持有(非家族经营)常加分 |
| 复制成本 | 反演分析：白酒易仿、陈年烈酒难仿；数十年研发与百年数据不可速成 |

## 行业结构与客户利益（Building Block 侧）
- **迷你垄断**：看客户决策时点的真实选择而非理论选择（烟草）。**局部垄断**：区域霸权、转换成本（剃刀-刀片）。
- **寡头**优于碎片化战场：找结构长期稳定、且稳定逻辑仍在的寡头；理性机制（不打折政策）是长期观的标志。
- 低进入壁垒 ≠ 低成功壁垒（餐饮业）；老行业仍由创始家族持有是行业耐久+靠留存收益内生增长的标志（糖果业）。
- 客户利益三类：**无形**（口味/身份，入口上脸的产品溢价潜力大）、**保证**（婴儿食品/检测，声誉无法用钱速成）、**便利**（易被就近的新对手侵蚀）。
- 期望产出（T2）：强而可预测的现金生成、可持续高资本回报、有吸引力的增长机会。
