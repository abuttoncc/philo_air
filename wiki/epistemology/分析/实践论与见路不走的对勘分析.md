# 实践论与见路不走的对勘分析

> 2026-06-12 主动编译产出。任务：将毛泽东《实践论》《矛盾论》与豆豆"见路不走"
> 进行系统对勘，填补知识库中"马克思主义认识论→豆豆方法论"的谱系缺口。

## 一、总判定：同一认识论内核的三层操作化

```
马克思主义认识论（马克思）
    ↓ 中国化转译
实事求是（毛泽东，1937）
    ↓ 可操作化
见路不走（豆豆/叶子农，2013）
    ↓ 决策论化
条件的可能（判断尺）+ 事于道（反题）
```

三层不是替代关系，而是**同一内核的逐级操作化**：
- 马克思：哲学原则（认识对实践的依赖）
- 毛泽东：政治方法论（调查研究、群众路线、矛盾分析）
- 豆豆：个体决策程序（条件评估、因果推断、反教条）

## 二、逐项对勘

### 2.1 反对教条主义

| 《实践论》 | 见路不走 |
|---|---|
| "纠正脱离实际情况的本本主义" | "不拘经验教条" |
| 教条主义 = 把理论当公式 | 事于道 = 把方法当法宝 |
| 认识论错误：理论脱离实际 | 认识论错误：方法脱离条件 |

**同构判定**：[[教条主义]] →instance_of→ [[事于道]]

### 2.2 反对经验主义

| 《实践论》 | 见路不走 |
|---|---|
| "尊重经验而看轻理论" | "路=成功者的经验" |
| "一孔之见" | "他那个条件的可能，你不可能完全复制" |
| 经验主义 = 把局部经验普遍化 | "路"迷信 = 把他人经验照搬 |

**同构判定**：经验主义与"路"迷信同构

### 2.3 真理标准

| 《实践论》 | 见路不走 |
|---|---|
| "依客观上社会实践的结果如何而定" | "走条件的可能"（以实际结果验证） |
| 社会实践 | 条件满足 |
| 集体验证 | 个体验证 |

**差异**：毛泽东的标准是**社会实践**（集体、历史），豆豆的标准是**条件满足**（个体、即时）。

### 2.4 具体分析

| 《矛盾论》 | 见路不走 |
|---|---|
| "具体地分析具体的情况" | "立足自身的条件，以自身的条件可能为判断" |
| 矛盾特殊性分析 | 条件特殊性评估 |
| 找出主要矛盾 | 找出关键约束条件 |

**同构判定**：矛盾分析 ↔ 条件分析，同一方法论框架的不同应用领域。

### 2.5 操作界面

| 毛泽东 | 豆豆 |
|---|---|
| 调查研究 | 条件评估 |
| 群众路线 | （无集体维度） |
| 矛盾分析法 | 因果推断 |
| 阶级分析 | 能力圈分析 |

**关键差异**：毛泽东的操作界面是**集体政治行动**；豆豆的操作界面是**个体理性决策**。

## 三、谱系定位

### 在知识库既有谱系中的位置

此前知识库已追踪的谱系线（[[文化属性论的理论沿革谱系]]）：

```
历史唯物主义（马克思）──→ 文化决定论论证（豆豆，方向倒转）
```

本次补充的新谱系线：

```
实事求是（毛泽东）──→ 见路不走（豆豆，可操作化）
    ↑ develops
《实践论》《矛盾论》（认识论-方法论经典）
```

**两条线的关系**：
- 线1（历史唯物主义→文化决定论）：**方向倒转**——借马克思主义话语，装入文化决定论内核
- 线2（实事求是→见路不走）：**继承发展**——明言继承，操作化升级

豆豆对马克思主义的双重姿态：
1. **《遥远的救世主》**：借用话语，倒转方向（文化决定论）
2. **《天幕红尘》**：正面认同，操作化发展（见路不走）

这一演化已在[[天道三部曲思想演化]]中记录，本次对勘为其提供了**认识论谱系的深层支撑**。

## 五、批判性评估

1. **操作化代价**：从"实事求是"到"见路不走"，政治维度被剥离，集体行动被还原为个体决策。
   这是进步（更可执行）还是退步（失去社会变革维度）？知识库保持开放。

2. **验证标准收窄**：从"社会实践"到"条件满足"，验证标准从集体历史检验收窄为个体即时检验。
   这提高了可操作性，但降低了抗错能力（个体判断失误无集体纠偏机制）。

3. **辩证法稀释**：从"矛盾斗争"到"条件约束"，辩证法的动态性被稀释为静态分析。
   豆豆体系是否因此失去了对"变化"的把握？[[事于道]]作为反题部分回应了这一点。
