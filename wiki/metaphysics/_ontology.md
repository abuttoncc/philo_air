# Philo Vault · 形而上学领域本体契约（本库参考母本）

> `wiki/metaphysics/` 的本体定义。`ingest`/`recall` 前先读本页。引擎规范见 `.claude/skills/auto-wiki/`，
> 6 档时间模型、退役不删除协议与三分铁律以引擎为准。
> **本页是 Philo Vault 的参考母本**：声明哲学域通用的节点类型判据与受控关系词表；
> epistemology / ethics 的 `_ontology.md` 只声明各自增量，其余沿用本页。

## 0. 总原则
1. **节点 / 数据 / 边 三分**：数值绝不是节点（进 data.db）；关系是边不是页；分类标签是边不是页。
2. **编译单向**：`Inbox(散文/讨论) → ingest → wiki(本体)`。严谨只用在已结晶的知识。
3. **退役不删除**：T1/T2/T3 任何变化都是「旧行封 valid_to + 插新行」，永不 DELETE，必有 T4 事件盖章。
   **个人立场（T1 facts）的切换轨迹是本库最有价值的时间序列，永不删旧行。**

## 1. 节点判据（「能不能指向就是这一个」）
能用手指指向「就是这一个」、明年同名还指同一个 → **实体**；必须先讲一段「它怎么定义」才听得懂 → **概念**。
> 哲学域易踩的坑：
> - 「电车难题」「中文房间」「葛梯尔反例」有具体场景、像实体——它们是**有专名的论证/思想实验**，归 `论证/`（介于实体与概念之间的第三类，判据：有专名 + 可指认出处 + 功能是支持/反驳某概念）。
> - 「物自体」「随附性」「可能世界」有严格定义、有符号记法，但仍是**概念**（没有「创设它的那一个专名物」）。
> - 「学界主流立场=物理主义」不是节点，是 **T1 fact**（facts 拉链表）。

## 2. 节点类型（=目录=图谱着色）

| 类型 | 子目录 | 判据 | 例子 |
|---|---|---|---|
| 实体·人物 | `人物/` | 有专名的哲学家/学者，能指向就是这一个 | 康德、亚里士多德、笛卡尔、刘易斯(David Lewis)、查尔默斯 |
| 实体·学派 | `学派/` | 有专名的历史/学术共同体（有成员、有时段） | 斯多亚学派、维也纳学圈、德国观念论 |
| 实体·著作 | `著作/` | 有专名的文本，书名号《》开头命名 | 《纯粹理性批判》《物质与意识》《命名与必然性》 |
| 概念 | `概念/` | 必须先讲一段定义/机理才懂；问题型概念（如心身问题）也归此，`classified_as` 问题 | 自由意志、物自体、随附性、因果、同一性、心身问题 |
| 论证 | `论证/` | 有专名的论证/思想实验：有出处、功能是支持/反驳某概念 | 知识论证(玛丽房间)、中文房间、弗兰克福案例、宇宙论论证 |
| 事件 | `事件/` | 有日期+施动者，发生即固定 | 1781-纯粹理性批判出版、1929-达沃斯之辩、2020-PhilPapers调查发布 |
| 分析 | `分析/` | 研究课题派生视图（删了不影响本体） | 自由意志与道德责任全景、AI意识问题综述 |
| 来源 | `来源/` | SEP/IEP 条目摘录、播客纪要、讲义、与 Agent 的讨论纪要 | 2026-06-11-SEP-自由意志条目、2026-06-12-关于相容论的对谈 |

三分铁律：
- **数值进 data.db** —— 「PhilPapers 2020 调查中相容论支持率=59%」不建页，进 `data_points`。
- **关系是边** —— 「康德写《纯粹理性批判》」是一条 `authored_by` 边。
- **分类标签是边** —— 大陆/分析、一元论/二元论、相容论/不相容论是贴在概念/人物上的 `classified_as` 边，可按它分桶着色，但不建页。

## 3. 受控关系词表（哲学域通用，本页为全库母本）

- `proposed_by`（概念/论证 → 人物）：提出/创设。例：物自体 →proposed_by→ 康德。
- `authored_by`（著作 → 人物）：作者。例：《纯粹理性批判》 →authored_by→ 康德。
- `belongs_to`（人物 → 学派）：归属。例：石里克 →belongs_to→ 维也纳学圈。
- `appears_in`（概念/论证 → 著作）：出处。例：知识论证 →appears_in→ 《副现象的感受质》。
- `responds_to`（论证 → 概念[问题型]）：回应某问题。例：弗兰克福案例 →responds_to→ 自由意志。
- `supports`（论证/人物 → 概念/论证）：支持。例：弗兰克福案例 →supports→ 相容论。
- `critiques`（论证/人物 → 概念/论证）：批评/反驳。例：中文房间 →critiques→ 强人工智能。
- `develops`（后继概念/人物 → 前驱概念/人物）：发展/细化。例：随附性物理主义 →develops→ 物理主义。
- `instance_of`（论证 → 概念）：类属。例：中文房间 →instance_of→ 思想实验。
- `part_of`（概念 → 概念）：构成。例：心身问题 →part_of→ 形而上学问题域。
- `classified_as`（→ 分类标签）：分类标签边（一元论/二元论、分析/大陆…），标签不建页。
- `created_by` / `changed_by`（→ 事件）：T4 事件盖章。
- **`grounds`（理论哲学节点 → 任意域节点，跨域允许）：奠基/蕴含。** 本方向签名边，打通理论→实践。
  例：自由意志怀疑论 →grounds→ [[道德责任]]（ethics）；外在主义 →grounds→ [[知识]]（epistemology）。
- `references`（分析/来源 → 任意）：只溯源。

> lint 时以本表对照；越界关系边拒绝。双写：页面 `frontmatter.relations[]` + `data.db relations 表`，两者一致。

## 4. 跨域连接（理论 ↔ 实践）
本域是实践哲学的奠基层。**优先用 `grounds`** 表达「形而上学立场 → 伦理/认识论后果」（强语义，可查询）；
纯溯源/弱关联用 `references`。跨域 wikilink 直接写对方域的 slug（如 [[康德]] 已在本域建页，
ethics 不重复建，直接连边引用——命名纪律：单一中文 slug 全库唯一，译名分歧记 `aliases:`）。

## 5. 时间档裁决示例（哲学域专用）
- 「我当前对自由意志的立场=相容论」→ **T1**（facts 拉链，立场切换必有 T4 事件盖章：哪次讨论/哪本书改变了我）。
- 「弗兰克福案例反驳可供取舍可能性原则」→ **T2**（耐用论证，论证页正文 + durability）。

- 「PhilPapers 2020 相容论支持率 59%」→ **T0**（data_points，逐次调查一条）。
