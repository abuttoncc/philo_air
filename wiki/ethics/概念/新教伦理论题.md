# 新教伦理论题（韦伯论题）

[[韦伯]]在[[《新教伦理与资本主义精神》]]（1904–05）中提出的命题：
**新教入世禁欲伦理与近代资本主义精神之间存在选择性亲和（Wahlverwandtschaft）**——
宗教伦理这一文化要素，参与解释了为何理性资本主义首先在西方新教地区系统性兴起。

## 论证链（转述概括）

```
加尔文宗预定论 → 救赎确证焦虑 → 入世禁欲（职业成功=蒙选的间接记号）
→ 系统性理性劳动 + 禁欲性限制消费 → 资本积累与再投资的伦理化
→ 资本主义"精神"（以营利为天职）
```

## 方法论身位（关键限定）

- **不是文化单因论**：韦伯结语明确拒绝以唯灵论单因取代唯物论单因（转述，出处：结语章）。
  论题主张的是文化要素的**不可还原贡献**，不是文化决定一切。
- **理念型方法**：「资本主义精神」「禁欲新教」都是理念型（Idealtypus），不是统计平均。
- **后续争议**：经济史学界对论题的经验效度长期争论（如对天主教地区资本主义的反例研究），
  但"文化要素参与塑造经济结果"的弱版本已成共识框架。

## 在本库谱系中的位置

本论题是[[文化属性]]（豆豆）的最近社会科学前驱——结构平行：

| | 韦伯论题 | 文化属性论 |
|---|---|---|
| 解释项 | 宗教伦理（文化要素） | 文化属性（价值判断体系） |
| 被解释项 | 资本主义兴起与否 | 个体/民族的命运强弱 |
| 因果强度 | 选择性亲和（多因之一） | 「归根到底」的强决定（第3章原文） |
| 方法 | 历史比较 + 理念型 | 格言式断言 + 小说情节验证 |
